4.3 Binomial Distribution

4.3 Phân phối nhị thức

X ~ B(n, p) means that the discrete random variable X has a binomial probability distribution with n trials and probability of success p.

X ~ B(n, p) nghĩa là biến ngẫu nhiên rời rạc X có phân phối xác suất nhị thức với n phép thử và xác suất thành công là p.

X = the number of successes in n independent trials

X = số lần thành công trong n phép thử độc lập

n = the number of independent trials

n = số lượng các phép thử độc lập

X takes on the values x = 0, 1, 2, 3, ..., n

X nhận các giá trị x = 0, 1, 2, 3, ..., n

p = the probability of a success for any trial

p = xác suất thành công cho bất kỳ phép thử nào

q = the probability of a failure for any trial

q = xác suất thất bại cho bất kỳ phép thử nào

p + q = 1

q = 1 – p

The mean of X is μ = np. The standard deviation of X is σ = √𝑛⁢𝑝⁢𝑞n⁢p⁢qnpq.

Số trung bình của X là μ = np. Độ lệch chuẩn của X là σ = √𝑛⁢𝑝⁢𝑞n⁢p⁢qnpq.

4.5 Hypergeometric Distribution

4.5 Phân phối siêu bội

X ~ H(r, b, n) means that the discrete random variable X has a hypergeometric probability distribution with r = the size of the group of interest (first group), b = the size of the second group, and n = the size of the chosen sample.

X ~ H(r, b, n) có nghĩa là biến ngẫu nhiên rời rạc X có phân phối xác suất siêu bội với r = kích thước của nhóm quan tâm (nhóm thứ nhất), b = kích thước của nhóm thứ hai, và n = kích thước của mẫu được chọn.

X = the number of items from the group of interest that are in the chosen sample, and X may take on the values x = 0, 1, ..., up to the size of the group of interest. (The minimum value for X may be larger than zero in some instances.)

X = số lượng các phần tử từ nhóm quan tâm có trong mẫu được chọn, và X có thể nhận các giá trị x = 0, 1, ..., lên đến kích thước của nhóm quan tâm. (Giá trị tối thiểu cho X có thể lớn hơn không trong một số trường hợp.)

4.6 Poisson Distribution

4.6 Phân phối Poisson

X ~ P(μ) means that X has a Poisson probability distribution where X = the number of occurrences in the interval of interest.

X ~ P(μ) có nghĩa là X có phân phối xác suất Poisson trong đó X = số lần xuất hiện trong khoảng quan tâm.

X takes on the values x = 0, 1, 2, 3, ...

X nhận các giá trị x = 0, 1, 2, 3, ...

The mean μ is typically given.
