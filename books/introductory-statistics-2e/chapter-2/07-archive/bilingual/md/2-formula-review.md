where *i* = the ranking or position of a data value,

trong đó *i* = thứ hạng hoặc vị trí của một giá trị dữ liệu,

*k* = the kth percentile,

*k* = bách phân vị thứ k,

*n* = total number of data.

*n* = tổng số lượng dữ liệu.

Expression for finding the percentile of a data value: (𝑥 + 0.5⁢𝑦𝑛)(x + 0.5⁢yn)(x + 0.5yn)(100)

Biểu thức để tìm bách phân vị của một giá trị dữ liệu: (𝑥 + 0.5⁢𝑦𝑛)(x + 0.5⁢yn)(x + 0.5yn)(100)

where *x* = the number of values counting from the bottom of the data list up to but not including the data value for which you want to find the percentile,

trong đó *x* = số lượng các giá trị đếm từ dưới cùng của danh sách dữ liệu lên đến nhưng không bao gồm giá trị dữ liệu mà bạn muốn tìm bách phân vị,

*y* = the number of data values equal to the data value for which you want to find the percentile,

*y* = số lượng các giá trị dữ liệu bằng với giá trị dữ liệu mà bạn muốn tìm bách phân vị,

*n* = total number of data

*n* = tổng số lượng dữ liệu
