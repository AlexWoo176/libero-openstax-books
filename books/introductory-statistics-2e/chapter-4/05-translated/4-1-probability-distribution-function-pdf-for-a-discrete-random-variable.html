4.1 Probability Distribution Function (PDF) for a Discrete Random Variable

4.1 Hàm phân phối xác suất (PDF) cho một biến ngẫu nhiên rời rạc

A discrete probability distribution function has two characteristics:

Một Hàm phân phối xác suất rời rạc có hai đặc điểm:

  1. Each probability is between zero and one, inclusive.
  2. Mỗi xác suất nằm trong khoảng từ không đến một, bao gồm cả hai giá trị này.
  3. The sum of the probabilities is one.
  4. Tổng các xác suất bằng một.

Ví dụ 4.1

X takes on the values 0, 1, 2, 3, 4, 5. This is a discrete PDF because:

X nhận các giá trị 0, 1, 2, 3, 4, 5. Đây là một PDF rời rạc vì:

  1. Each P(x) is between zero and one, inclusive.
  2. Mỗi P(x) nằm trong khoảng từ không đến một, bao gồm cả hai giá trị này.

  4. Tổng các xác suất bằng một, nghĩa là,
250+1150+2350+950+450+150=1250+1150+2350+950+450+150=1250+1150+2350+950+450+150=1
